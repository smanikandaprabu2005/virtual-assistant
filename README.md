#  Virtual Assistant (Shifra)

Virtual Assistant Shifra is a browser-based voice assistant built using JavaScript, Web Speech API, and Speech Synthesis API.  
It listens to voice commands, responds with speech, and performs basic tasks like opening websites, telling the time and date, and searching the internet.

##  Features

- Voice recognition using SpeechRecognition API
- Text-to-speech responses using SpeechSynthesis
- Greets the user based on time
- Tells current time and date
- Opens popular websites using voice commands
- Handles microphone and speech errors
- Fully client-side application (no backend)

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Web Speech API
  - Speech Recognition
  - Speech Synthesis

##  Supported Voice Commands

| Command | Action |
|-------|--------|
| hello / hey | Greets the user |
| who are you | Introduces the assistant |
| your name | Speaks assistant name |
| how are you | Conversational reply |
| open youtube | Opens YouTube |
| open google | Opens Google |
| open facebook | Opens Facebook |
| open instagram | Opens Instagram |
| open calendar | Opens Google Calendar |
| time | Speaks current time |
| date | Speaks current date |
| other queries | Google search |
